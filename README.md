# 🍽️ Food Menu App

A simple React-based Food Menu application that allows users to search and explore dishes.

## 🚀 Features

* 🔍 Search functionality (case-insensitive)
* 🧠 Multi-word filtering
* 📱 Responsive UI
* ➕ Add button for each item

## 🛠️ Tech Stack

* React.js
* CSS

## 📂 Project Setup

```bash
npm install
npm run dev
```
